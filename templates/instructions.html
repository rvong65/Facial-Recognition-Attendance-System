Step 1

Create a folder with the images of faces. Be sure to name each image with the name of the person like this: Chris Evans. Click on the Choose file button and select all the images using your mouse cursor. Click on submit when you are done.

Step 2

This step requires a webcam in order to recognize the faces. Adjust the webcam in a way in which it will be able to see the faces of people. When a face is detected, it will output a box with their name like in this picture:

Step 3

When you are finished, click the download button and you will receive a text file with the names and times of those recorded along with the names of those who are absent.
